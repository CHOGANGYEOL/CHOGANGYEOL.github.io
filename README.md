# Git Commit Convention

- 기본 형식 : [commit type]: [commit message]
- commit type
  - feat : 기능 구현
  - fix : 버그 수정
  - docs : 문서 작업
  - refactor : 리팩토링
  - test : 테스트
  - chore : 기타 작업
